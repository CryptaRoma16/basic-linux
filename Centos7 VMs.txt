2. CREATE A NEW VIRTUAL MACHINE
3. CUSTOM (ADVANCED) -> NEXT -> I WILL INSTALL THE OPERATING SYSTEM LATER
4. LINUX -> VERSION : CENTOS 7 OR 8 64-BIT (RUN CMD: systeminfo)
5. VIRTUAL BASE INSTALL: CENTOS BASE INSTALL FIRST
6. NUMBER OF PROCESSORS: 2, NUMBER OF CORES PER PROCESSOR: 2, TOTAL PROCESSOR CORES: 4
7. MEMORY FOR THE VIRTUAL MACHINE: 4GB
8. NETWORK TYPE: USE BRIDGED NETWORKING (TO GET UNIQUE IP ADDRESS)
9. LSI LOGIC (RECOMMENDED) - I/O CONTROLLER TYPES
10. SCSI (RECOMMENDED) - VIRTUAL DISK TYPE
11. CREATE VIRTUAL DISK
12. MAXIMUM DISK SIZE (GB): 20.0 GB, SPLIT VIRTUAL DISK INTO MULTIPLE FILES
13. DISK FILE: .vmdk -> NEXT -> FINISH
